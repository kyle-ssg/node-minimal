Simple Node Server
==================================

- CORS support via [cors](https://github.com/troygoode/node-cors)
- Body Parsing via [body-parser](https://github.com/expressjs/body-parser)

Getting Started
---------------

# Setup via cli
```$ npm i ssg-node -g```
```$ ssg-node PROJECT_NAME```

# Run
```$ npm start```

# Nodemon
```$ npm run dev```

# Automated Tests
```
    npm run tests
```
